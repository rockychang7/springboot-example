## springboot构建Test组件基于依赖
- JUnit 5：单元测试的一个标准框架
- Spring Test & Spring Boot Test：支持整合springboot的基础框架
- AssertJ： 断言框架
- Hamcrest：一个匹配器对象库（断言）
- Mockito： 一个Java mock库
- JSONassert：用于Json的断言库
- JsonPath： Xpath框架
