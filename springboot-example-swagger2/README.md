## SpringBoot整合swagger(springfox3)
 


### 依赖
```
<dependency>
    <groupId>io.springfox</groupId>
    <artifactId>springfox-boot-starter</artifactId>
    <version>3.0.0</version>
</dependency>
```
### 注解
通过@EnableOpenApi开启

### 运行路径
通过`http://localhost:8080/swagger-ui/index.html`进入登录界面
